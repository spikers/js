Conclusion

It looks like window.setInterval starts the counter immediately after the previous counter ends.

So let's say we have an operation that takes 5 seconds. Timeout is 1 minute. Start: 00:00:00 in hh:ii:ss. It'll run the set interval at 00:01:00, 00:02:00, 00:03:00, as opposed to 00:01:00, 00:02:05, 00:03:10. Data to back it up:
